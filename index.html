Oral History as Data is a static web tool that allows you to publish and analyze oral histories and/or qualitative interviews online for free using GitHub Pages and Jekyll.

This site and the GitHub Repository it's built on can be used as a template for producing your own digital collection of searchable and visualized transcripts.
